# Java-Demos

A Spring Boot REST API backend project for learning and demonstrating advanced Spring Boot concepts and best practices.

## Overview

This repository serves as a hands-on learning platform for implementing various Spring Boot features and patterns, focusing on real-world scenarios and challenges.

## Learning Objectives

- **Exception Handling**: Implementing global exception handlers and custom error responses
- **Database Concurrency**: Managing concurrent database updates and handling optimistic/pessimistic locking
- **Multithreading**: Working with concurrent operations, thread pools, and async processing
- **Transaction Management**: Ensuring data consistency in multi-threaded environments

## Technologies

- Java
- Spring Boot
- Spring Data JPA
- Maven/Gradle (specify your build tool)
